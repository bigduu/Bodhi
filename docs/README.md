# Bodhi Docs

This directory contains documentation for the Bodhi desktop shell (`src-tauri`) and release/runtime concerns.

## Scope Boundary

- Bodhi owns desktop shell runtime, packaging, release, and integration glue.
- Lotus owns frontend product implementation and frontend-focused documentation.
